The Data Behind the Map

Use this tool to explore and manage the data behind the DAI at a glance map. You can generate the GeoJSON used to place the pins on the map, the CartoCSS used to shade active and inactive countries—or generate a host of more human-readable views of the data, like an outline of all country data for countries with active projects.

All data is pulled from the DAI Country Data YAML file on Github, which is updated via Sharepoint, Oracle, and ocassionally by hand—as needed.

Generate Data Files

Point GeoJSON

This will generate the GeoJSON used to place pins on the map. Properties include description, marker-color, symbol, size, title, and coordinates.

Active Shades CartoCSS

This will generate the CartoCSS used to shade countries active (dark-green) or inactive (light-green)—based on whether or not any active projects exist within the country object.

USAID Shades CartoCSS

This will generate the CartoCSS used to shade countries with active projects from USAID.

DFID Shades CartoCSS

This will generate the CartoCSS used to shade countries with active projects from DFID.

EC Shades CartoCSS

This will generate the CartoCSS used to shade countries with active projects from EC.

Generate Human Readable Views

Active Country Data Outline

All country data for countries that contain active projects. Properties include country name, code, project name, acronym, client, start date, end date, dai.com url.
